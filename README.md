Recentemente iniciei na vaga de Assitente Web na Universidade Iguaçu e estou muito feliz com minha conquista. Como parte de meus afazeres, a criação de automações para processos internos é uma delas, nesse repositório postarei o desenvolvimento de um pequeno software para a resolução desses processos.

No momento estou focado em desenvolver a interface que irá suportar essas automações, mais para frente o foco será em bibliotecas como pyautogui e selenium para alcançar meus objetivos.

Funcionalidades: Navegação entre janelas, Interface intuitiva, Receptaculo de futuras automações.

Tecnologias utilizadas: Python3, CustomTinter, Pillow e Pathlib.
